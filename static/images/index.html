      The paper titled "Pneumonia detection using Convolutional Neural Networks: A Deep Learning approach" proposed a novel CNN-based model for detecting pneumonia.       Brain Tumour Detection using Machine Learning

        The research paper "Brain Tumor Detection Using Machine Learning Techniques" proposed a machine learning-based approach for detecting brain tumors. The model used a combination of wavelet transform and principal component analysis to extract features from MRI images. The approach achieved high performance on a dataset of 60 MRI images.
